Consumer app demonstrating how to use core_api_client_package.

Run:
1. cd example_consumer
2. flutter pub get
3. flutter run

This app initializes ApiClient with baseUrl https://jsonplaceholder.typicode.com and fetches /todos/1 when you press the button.

The dependency is a local path to the package (path: ..). For production, replace with a git dependency or a pub.dev version.
